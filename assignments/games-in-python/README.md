
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You'll create a Hangman game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Create the Word Selection System

#### Description
Implement the functionality to select a random word from a predefined list that players will try to guess throughout the game.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Store the selected word for the game session
- Support multiple words in the list for variety

### 🛠️ Implement Guess Processing and Display

#### Description
Build the system to accept letter guesses from players and display the current progress of the hidden word.

#### Requirements
Completed program should:

- Accept letter guesses from the player
- Show current progress using underscore format (_ _ _)
- Track correct and incorrect guesses
- Prevent duplicate guess handling

### 🛠️ Add Game Flow and Win/Loss Conditions

#### Description
Create the main game loop that manages attempts, determines win/loss conditions, and displays appropriate messages.

#### Requirements
Completed program should:

- Track incorrect guesses remaining
- End game when word is guessed or attempts exhausted
- Display clear win/lose messages
- Handle game completion gracefully
